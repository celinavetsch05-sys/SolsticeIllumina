# Solstice Illumina

Personal website for Celina Vetsch. A single-page HTML/CSS/JS site with a cosmic, mystical aesthetic.

## Project Structure

- `index.html` — the entire site (HTML, CSS, and JS all in one file)
- Deployed via Vercel, source on GitHub at `celinavetsch05-sys/SolsticeIllumina`

## Design System

| Token | Value | Role |
|-------|-------|------|
| `--void` | `#04051a` | Background |
| `--solstice` | `#6b3fa0` | Primary purple |
| `--lumina` | `#c4a8ff` | Light purple / accents |
| `--gold` | `#d4a847` | Gold highlights |
| `--iris` | `#8de8d8` | Teal accents |
| `--stardust` | `#f0ecff` | Body text |
| `--muted` | `#7a6e9a` | Muted text |

Font: Georgia, serif

## Conventions

- All styles, markup, and scripts live in `index.html` — do not split into separate files unless asked
- Preserve the custom cursor and trail-dot effects
- Keep the cosmic / solstice aesthetic when adding new sections or components
- After any change, commit and push to GitHub so Vercel auto-deploys

## Plugins Installed

- `superpowers` — planning, brainstorming, debugging, code review workflows
- `ui-ux-pro-max` — UI/UX design intelligence (use for design decisions, color, typography)
